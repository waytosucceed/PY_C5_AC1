String Methods

Methods are functions associated with an object or class in Python Programming. They are used to perform specific tasks or operations on the data stored within objects or classes. In Python, all variables are objects. We will learn about the class in upcoming classes.

In simple words, methods are actions that an object can perform. You use a method by attaching it to an object with a dot (.). Different objects have different methods that can be performed on them. Let’s have a look at methods we can perform on strings:

  • lower() - Converts all characters into lowercase
  • upper() - Converts all characters into uppercase
  • strip(), lstrip(), rstrip():
    • strip() - Removes space at the beginning and end of a string
    • lstrip() - Removes space from the beginning of the string
    • rstrip() - Removes space at the end of the string
  • replace() - Replaces a specified word with another word
  • find() - Finds the first occurrence of a word in a string.

    Returns 0 if word is found.
    Returns -1 if word not found.

format():

f-strings:

  • f-strings allow you to directly embed expressions inside string literals using curly braces ({}).
  • f-strings were introduced in Python 3.6.

Strings in Python are immutable, which means you cannot change a string once it is created. Any modifications we do using string methods result in a new string.
